Instructions
How to use proxy configuration and important notes

Proxy will be applied to the following operations:

  • Token refresh (Social/Builder ID/IdC)
  • Account usage queries
  • Chat requests (streaming and non-streaming)
  • Available models list

ISP proxy format example:

38.111.61.65:443:NNDyNdwnJkFt:ykWE3acywd

Standard HTTP proxy format:

http://username:password@proxy.example.com:8080
